# Naor Shem-Tov - Data & AI Portfolio

A bilingual portfolio website showcasing independent Data, AI, and BI projects, professional experience, education, and selected achievements.

## Local development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Editing content

- `app/content.ts` - English and Hebrew content
- `app/portfolio-page.tsx` - page structure
- `app/globals.css` - visual design and responsive styles

## Deployment

The project is configured for deployment on Vercel as a Next.js application.
